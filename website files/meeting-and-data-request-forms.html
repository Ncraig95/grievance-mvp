Meeting and Data Request Forms

These forms are mainly used after a grievance record already exists. Have the grievance number, member name, contract, dates, and representative information ready before starting.

Important: If any form will not open, will not submit, sends you to the wrong place, or does not create the expected document, contact the hall immediately.

BellSouth Meeting Request

Use this form when: A BellSouth grievance needs a meeting request created against an existing grievance folder.

How to fill it out: Enter the existing grievance number, grievant information, occurrence date, issue or contract section, informal meeting date, requested meeting date, time, place, union representative, company representatives, and any extra information.

What it should do: The system generates the BellSouth meeting request and stores it with the existing grievance record.

Mobility Meeting Request

Use this form when: An AT&T Mobility grievance needs a meeting request created against an existing grievance folder.

How to fill it out: Enter the existing grievance number, grievant information, occurrence date, issue or contract section, informal meeting date, requested meeting date, time, place, union representative, company representatives, and any extra information.

What it should do: The system generates the Mobility meeting request and stores it with the existing grievance record.

Grievance Data Request

Use this form when: The union needs to request information from the company for an existing grievance.

How to fill it out: Enter the grievance number, contract, grievant name, articles involved, company name, company representative name and title, due date, union phone, union representative name, and union representative title.

What it should do: The system generates a grievance data request and files it in the existing grievance folder.

Data Request Letterhead

Use this form when: The union needs a data request cover letter on letterhead for an existing grievance.

How to fill it out: Enter the grievance number, contract, grievant name, company representative name, title, email if available, the records or data requested, preferred format, steward name, steward email, and optional approval block.

What it should do: The system generates the data request letterhead document and files it in the existing grievance folder.
